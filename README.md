### Requirements
- Python >= 3.8
- PyTorch >= 1.9
- Pytorch Lightning >= 1.4.5
- CUDA enabled computing device

### Installation
```
$ git clone https://github.com/vrvrv/PyTorch-SBM
$ cd PyTorch-SBM
$ pip install -r requirements.txt
```
